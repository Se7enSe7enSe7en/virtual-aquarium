# virtual-aquarium
A virtual aquarium made with vue

## Git hub page
https://se7ense7ense7en.github.io/virtual-aquarium/

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```

### Build and Deploy git hub page
```sh
pnpm run deploy
```
